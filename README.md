# Tienda-Virtual
Tienda Virtual de motos

ESTE FUE EL PRIMER PROMPT QUE HSIMOS ESTE FUE EL QUE USAMOS PARA GENERAR EL HTML DEL CLIENTE:

CONTEXTO:

Iron Rebel Garage es una tienda de motocicletas que necesita el diseño de su catálogo digital inicial. La marca busca posicionarse con una identidad propia dentro de un mercado dominado por referentes como Ducati, BMW Motorrad, Harley-Davidson y Yamaha, La sensación que debe transmitir la página es potencia, exclusividad y adrenalina.

ROL:

actua como un programdor de sofware que quiere diseñar una pagia de motos Tu tarea es diseñar la estructura y el sistema visual completo del catálogo digital de Iron Rebel Garage.

CARACTERÍSTICAS:

Alcance del catálogo:
Entre 10 y 20 modelos de motocicleta.
Variedad real de estilos: Naked, Cruiser, Sport, Adventure, Urban, Touring.
Variedad real de cilindraje: desde motos de bajo cilindraje hasta motos de alto rendimiento
REQUISITOS:

Barra de navegación
Crear una barra de navegación completamente funcional, moderna, rápida y lista para integrarse en un sitio web profesional, con todos los filtros operativos y una excelente experiencia de usuario.

Sección "Encuentra tu moto" (filtro interactivo)
configurar un catálogo de motocicletas completamente funcional, visualmente atractivo, profesional y optimizado, con filtros avanzados totalmente operativos, búsqueda en tiempo real y fichas detalladas que destaquen aspectos como la cilindrada, potencia, rendimiento, seguridad y comodidad de cada motocicleta, ofreciendo una excelente experiencia de usuario y una integración perfecta con la barra de navegación y el sistema de filtros.

Clave para organizar los 10-20 modelos sin mostrarlos como lista.
Grid de catálogo (tarjetas de producto)

Imagen grande, nombre del modelo, 2-3 specs clave visibles, botón "Ver más".
Efecto hover premium (leve rotación o cambio de ángulo de la moto).
Grid visual, no tabla de datos cruda.
Página de detalle de modelo

Cada motocicleta debe presentar información técnica, nivel de comodidad, equipamiento y uso recomendado de forma clara, permitiendo al usuario filtrar, buscar, ordenar y comparar modelos de manera rápida y sencilla.

Footer

Redes sociales, newsletter, links legales, ubicación de tiendas físicas.
OBJETIVO
Presentar el catálogo de Iron Rebel Garage de forma atractiva, variada y profesional, generando deseo de compra y motivando al usuario a explorar los modelos y realizar una acción concreta de contacto o compra.

ESTE ES NUESTRO SEGUNDO PROMPT QUE USAMOS PARA GENERAR LA PARTE VISUAL-(CSS) PARA EL CLIENTE:

ROL:

Actúa como un Diseñador UX/UI Senior, especialista en interfaces premium para la industria automotriz y motocicletas, con experiencia en Glassmorphism (Vidrio Esmerilado), Neumorphism, diseño responsivo.

Tu objetivo es crear una interfaz visualmente impresionante, elegante, moderna y totalmente funcional, priorizando la experiencia del usuario y el rendimiento.

CONTEXTO:

Se desarrollará una página web para un catálogo profesional de motocicletas, donde los usuarios podrán explorar diferentes modelos, compararlos y encontrar la motocicleta ideal según sus necesidades.

Se necesita diseñar una plataforma web dedicada a la venta y exploración de motocicletas.

La página debe transmitir innovación, velocidad, confianza y tecnología, ofreciendo una experiencia moderna y atractiva para cualquier usuario.

El diseño debe ser completamente responsive y adaptarse perfectamente a:

Computadoras Laptops Tablets Celulares

La interfaz debe inspirarse en plataformas modernas como Apple, Tesla, Nothing, BMW Motorrad, Ducati y Yamaha.

El objetivo es ofrecer una experiencia visual limpia, elegante y altamente intuitiva.

RESULTADO ESPERADO:

Genera una interfaz completamente funcional y coherente, con un diseño de nivel profesional, utilizando el estilo Vidrio Esmerilado (Frosted Glass / Glassmorphism) combinado con Neumorphism, optimizada para cualquier dispositivo (móvil, tablet y escritorio). Cada componente debe mantener una identidad visual consistente, con animaciones suaves, excelente rendimiento, accesibilidad y una experiencia de usuario intuitiva. El resultado debe parecer una plataforma moderna de alta gama para la exploración y comparación de motocicletas.
